Practical Round Questions & React Tasks

Practical Round Questions

Question 1 – Login Validation

Create a Login Form

  • Email
  • Password

Requirements:

  • If email is empty → Show "Email Required"
  • If password is less than 6 characters → Show error
  • If both are valid → Display "Login Successful"

Question 2 – Product Price Calculator

Create a Form

  • Product Name
  • Price
  • Quantity

Requirements:

  • Calculate Total = Price × Quantity
  • Display Total Amount
  • If Total > 5000 → Show "Discount Eligible"

Question 3 – Student Grade System

Create a Form

  • Student Name
  • Mark

Requirements:

  • Mark ≥ 90 → Grade A
  • Mark ≥ 75 → Grade B
  • Mark ≥ 50 → Grade C
  • Below 50 → Fail
  • Display Result

Question 4 – Employee Salary Bonus

Create a Form

  • Employee Name
  • Salary

Requirements:

  • Salary > 30000 → Bonus 5000
  • Salary > 20000 → Bonus 3000
  • Otherwise → Bonus 1000
  • Display Bonus and Final Salary

Question 5 – Todo Task Counter

Create a Todo Application

Requirements:

  • Add Task
  • Display Task List
  • Show Total Tasks Count
  • Delete Task
  • Show "No Tasks Available" when list is empty

Question 6 – User Search

API: https://jsonplaceholder.typicode.com/users

Requirements:

  • Fetch all users

  • Add Search Box
  • Filter users based on entered name
